# Memoneo master API
Acts as a middleware between the Memoneo apps and the GraphQL API (Hasura).

## Env variables
```bash
DB_STORE=data
DB_USER=memoneo
DB_PASSWORD=...
DB_HOST=100.100.100.100
DB_PORT=5432
HASURA_SECRET_KEY="..."
HASURA_GQL_ENDPOINT_URL=https://hasura.domain.com/v1/graphql
AUTH_URL=http://localhost:8089
PORT=8088
```
